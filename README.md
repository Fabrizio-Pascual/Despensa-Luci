Hace 25 días arranqué un proyecto para ayudar a mi mamá con su almacén — y hoy ya está en producción 👇

Despensa Luci es la digitalización completa de su negocio: catálogo online, checkout, panel de administración y hasta el "fiado" de toda la vida, ahora llevado desde una app en vez de un cuaderno.

Lo hice solo, de punta a punta. Usé v0 para levantar rápido la base de datos, y a partir de ahí el trabajo fue mío: ajustes de backend cuando hicieron falta, y toda la capa de front-end (que es mi especialidad) — diseño, componentes, flujo de compra y panel de admin.

El desafío más grande no fue una pantalla, fue un dato: el stock. Con clientes comprando online y venta al mostrador al mismo tiempo, tenía que asegurarme de que nunca se desincronice — nada de vender de más ni mostrar productos que ya no están.

Stack: Next.js · React · TypeScript · Supabase · Tailwind CSS · Vercel
